# Joshua & Cleo's Bali Coaching Container Itinerary

A beautiful, responsive itinerary website for a transformative 24-day journey through Bali and Lombok (January 7-31, 2025).

## Features

- **Stunning Hero Section** with gradient overlay and background image
- **Responsive Design** that works beautifully on all devices
- **Interactive Elements** including scroll animations and hover effects
- **Detailed Itinerary** with two main destinations:
  - Ubud, Bali (January 7-18)
  - Lombok (January 18-31)
- **Activity Cards** with images and detailed information
- **Accommodation Details** with photo galleries
- **Smooth Scrolling** and modern animations

## Customization

### Images
All images are currently sourced from:
- Airbnb listings (for accommodations)
- Unsplash (for activities)

To add custom images:
1. Open `index.html`
2. Find the `<img src="...">` tags
3. Replace the URLs with your own image URLs or local file paths

### Colors
The color scheme uses a tropical palette. To change colors, edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2c5f2d;      /* Forest green */
    --secondary-color: #97c593;     /* Light green */
    --accent-color: #d4a373;        /* Warm gold */
    --ocean-blue: #4a90a4;          /* Ocean blue */
    --warm-sand: #f4e8d8;           /* Sand beige */
}
```

### Content
To modify dates, activities, or descriptions:
1. Open `index.html`
2. Find the relevant section
3. Edit the text directly
